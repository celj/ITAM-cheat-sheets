Vectors are always columns, but it is customary to write them as rows to save space.

Definition 1.1 (Homogeneous system) If \(\mathrm{b} = 0_{m\times 1}\), the system is said to be homogeneous and there is at least one solution, given by \(\mathrm{x} = 0_{n\times 1}\). \[ \text{i.e.}\quad A\mathrm{x} = 0. \]
